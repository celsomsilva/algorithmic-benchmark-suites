---

The Algorithmic Benchmark Suite (R) repository: [Algorithmic R repository](https://github.com/celsomsilva/algorithmic-benchmark-suite-R) 



A framework for implementing and benchmarking classical algorithms (Binary Search, Liked List, Dynamic Programming, Strings, Number Theory, etc.) in R, with automated testing, performance profiling, and reproducible reports.
